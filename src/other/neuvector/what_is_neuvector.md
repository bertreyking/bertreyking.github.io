# 什么是 NeuVector ？

NeuVector 是一款专门为容器安全而设计的安全平台，提供全面的容器安全防御和威胁检测。它使用了一系列高级技术，如深度学习和情境感知等，来识别和预防各种安全威胁，并在发现异常行为时进行及时报警和响应。

- 该平台特点
  1. 对容器内部对网络流量进行监控和分析
  2. 自动检测和阻止恶意行为和攻击，确保容器环境的安全性，漏洞管理等
  3. 对容器运行时防御、容器镜像扫描和安全策略管理
  4. 可视化的安全策略管理工具，使用户更好地管理和调整安全策略

NeuVector 支持多种容器平台

- Docker
- Kubernetes
- OpenShift
